# **CMS Hospital Quality Data Sources**
Comprehensive reference for all raw datasets used in the CMS Hospital Quality Analytics Project.

**Author:** Waldo Ketonou  
**Last Updated:** April 2026  
**Source:** Centers for Medicare & Medicaid Services (CMS) Provider Data Catalog  
**Purpose:** Ensure transparency, reproducibility, and authoritative data provenance for all datasets used in this project.

---

## **Overview**
This project uses publicly available datasets from the CMS Provider Data Catalog.  
Each dataset is linked to both its **official CMS dataset page** and its **direct CSV download link**.  
Access dates are included to support reproducibility, as CMS updates these datasets regularly.

---
