# R Scripts — CMS Hospital Readmission Analytics  
This folder contains the complete, end‑to‑end R workflow used to prepare data, build models, evaluate performance, and run scenario simulations for the CMS Hospital Readmission Analytics project.

Each script is modular, reproducible, and aligned with a unified modeling pipeline designed for clarity, interpretability, and recruiter‑ready presentation.

---

## 📁 R Pipeline Overview

### **01_healthcare_aligned_eda.R**  
Exploratory Data Analysis (EDA) aligned to healthcare quality metrics.  
Includes distribution checks, missingness review, variable profiling, and early insights that inform modeling decisions.

### **02_modeling_dataset.R**  
Builds the modeling dataset used across all downstream scripts.  
Handles cleaning, feature engineering, factor preparation, and creation of the final train/test‑ready dataset.

### **03_cms_linear_model.R**  
Implements the Linear Regression model.  
Outputs RMSE, MAE, R², adjusted R², predictions, and a clean results object for evaluation.

### **04_cms_random_forest.R**  
Implements the Random Forest model using `ranger`.  
Generates predictions, variable importance, and performance metrics for comparison against the linear model.

### **05_cms_model_evaluation.R**  
Consolidates baseline, linear regression, and random forest results.  
Produces comparison tables, ranking outputs, visualizations, and Tableau‑ready exports.

### **06_scenario_simulation.R**  
Runs What‑If analysis using both models.  
Applies controlled scenario changes, generates predictions, computes deltas, and exports detailed + summary tables for Tableau dashboards.
